# Königslieder

A mobile first webpage for the KIT gymnastics sportsclub 🤸‍♀️

- <https://www.königslieder.de/>
- <https://www.koenigslieder.de/>

## Tech Stack

- TypeScript
- React
- Next.js

## Getting Started

Run development server

```shell
yarn dev
```

Run production server

```shell
yarn start
```
